# Vue+Webpack脚手架
## 安装的依赖：
- [x] 1. vue
- [x] 2. vue-resource
- [x] 3. vue-router
- [x] 4. vue-router-transition
- [x] 5. vuex
- [x] 6. vux

## 特点：
1. 基于Vue-cli生成
2. 初始化实现路由和路由过渡动画
3. 基于ES6
4. .vue单文件开发

## 安装：
1. 安装node.js[https://nodejs.org]
2. clone本项目
3. cd项目目录
4. $ npm install

## 使用：
dev: $ npm run dev

build: $ npm run build
